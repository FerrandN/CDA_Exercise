Quelques bonnes pratiques d'accessibilité

Images décoratives

utiliser des attributs "alt" vides ou nuls pour les images purement décoratives afub d'éviter qu'elles soient lues par les technologies d'assistance.

Images informatives

Utiliser des descriptions appropriées dans les attributs "alt" pour les images informatives afin de fournir des informations équivalentes pour les utilisateurs qui ne peuvent pas voir les images. Les attributs title et aria-describedby ne sont pas indispensables !

Balises HTML sémantiques (image informative)

Utiliser des balises HTML sémantiques appropriées (telles que < header >, < nav >, < main >, < section >, < article >, < aside >, < footer >) pour améliorer la stricture et la compréhension du contenu par technologies d'assistance.

Accès au clavier (image informative)

S'assurer que toutes les fonctionnalités du site sont accessibles via le clavier pour les utilisateurs qui ne peuvent pas utiliser la souris.

Contraste des couleurs (image informative)

Utiliser des combinaisons de couleurs offrant un contraste suffisant pour rendre le contenu lisible pour les personnes atteintes de déficience visuelle.

Formulaire sémantiques (image informative)

Utiliser des balises HTML sémantiques pour les formulaires (telles que < form >, < input >, < label >, < select >, < textarea >, etc...) pour l'améliorer l'accessibilité et la compréhensible du contenu par les technologies d'assistance.
